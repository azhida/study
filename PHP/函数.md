###### array_flip 键值对互换
```
$a = ['a', 'b', 'c', 'd'];
$b = array_flip($a);
print_r($b);
# 输出
[
    'a' => 0,
    'b' => 1,
    'c' => 2,
    'd' => 3,
]
```

###### array_slice 
```
# 定义：在数组中根据条件取出一段值，并返回
array_slice(array,offset,length,preserve)

$a=array(0=>"Dog",1=>"Cat",2=>"Horse",3=>"Bird");
print_r(array_slice($a,1,2));
```
