###### 数据类型
- 字符串：string，str，用引号括起来的文本
- 整数：integer，int，普通的整数数字
- 浮点数：float，带小数点的数

###### 算术运算符
|运算符|表示|例子|
|:------:|:----:|:----:|
|+|加|1+1输出结果2|
|-|减|4-4输出结果0|
|*|乘|3*3输出结果9|
|/|除|2/5输出结果0.4|
|%|取模--返回除法的余数|7%2输出结果1|
|//|取整除--返回商的整数部分|7//2输出结果为3|
|**|幂--返回x的y次幂|3**4结果为3的4次方|

###### 简单运算优先级口诀
```
从左到右顺着来
括号里的优先算
乘除排在加减前
```
