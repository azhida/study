## 使用方法

### 安装步骤

1. VS Code插件安装
   ```
   1. 打开 VS Code
   2. 转到扩展商店
   3. 搜索 "AWS Toolkit"
   4. 安装并启用CodeWhisperer
   ```

2. 账号配置
   ```
   1. 登录AWS账号
   2. 配置IAM权限
   3. 选择使用区域
   ```

### 基本用法

1. 代码补全
   - 自动显示建议
   - Tab键接受建议
   - Alt+[ 或 Alt+] 切换建议

2. AWS服务集成
   - 自动补全AWS SDK代码
   - 生成服务配置
   - 提供最佳实践

3. 安全扫描
   - 实时代码分析
   - 漏洞提醒
   - 修复建议
