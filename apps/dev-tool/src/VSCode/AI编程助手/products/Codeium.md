# Codeium

## 产品信息

- 名称：Codeium
- 开发商：Exafunction
- 官网：https://codeium.com/
- 产品定位：高性能免费AI编程助手
- 技术特点：基于自研大型代码语言模型

## 使用方法

### 安装步骤

1. VS Code插件安装
   ```
   1. 打开 VS Code
   2. 转到扩展商店
   3. 搜索 "Codeium"
   4. 点击安装
   ```

2. 账号配置
   ```
   1. 注册Codeium账号
   2. 在IDE中登录
   3. 开始使用
   ```

### 基本用法

1. 代码补全
   - 自动触发补全
   - Tab键接受建议
   - 方向键切换建议

2. 代码生成
   - 通过注释描述需求
   - 自然语言命令
   - 快捷键触发

3. 代码解释
   - 选择代码
   - 请求解释
   - 生成文档
