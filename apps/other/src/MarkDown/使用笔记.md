## PHPStorm 中使用 MarkDown

以插件形式安装，支持数学公式可见即可得

1. File --> Settings -->  Plugins
2. 搜索 MarkDown 并选择 `MarkDown Editor` 进行安装
3. 点击 Apply 立即应用 即可。
