# docker compose 配置详解

- docker-compose.yaml

```yaml
version: '3.8'

services:

  ubuntu:
    build:
      dockerfile: ./Dockerfile.ubuntu # 指定 dockerfile
    # image: ubuntu:latest # 指定镜像名称
    container_name: my-ubuntu # 指定容器名称
    working_dir: /var/www/
    ports:
      - "8080:3000"
    restart: unless-stopped
    tty: true

```
