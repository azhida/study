# 使用笔记

## 实现睡眠效果

```js
// app.js

function sleep(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function demo() {
  console.log('开始');
  await sleep(5000); // 等待5秒
  console.log('5秒后');
}

demo().then(() => console.log('继续执行'));
```

也可以借助第三方库来实现：unicorn-magic ，atomic-sleep 等
