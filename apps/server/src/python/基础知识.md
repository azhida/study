###### 数据类型
- 字符串：string，str，用引号括起来的文本
- 整数：integer，int，普通的整数数字
- 浮点数：float，带小数点的数

###### 算术运算符
|运算符|表示|例子|
|:------:|:----:|:----:|
|+|加|1+1输出结果2|
|-|减|4-4输出结果0|
|*|乘|3*3输出结果9|
|/|除|2/5输出结果0.4|
|%|取模--返回除法的余数|7%2输出结果1|
|//|取整除--返回商的整数部分|7//2输出结果为3|
|**|幂--返回x的y次幂|3**4结果为3的4次方|

###### 简单运算优先级口诀
```
从左到右顺着来
括号里的优先算
乘除排在加减前
```

###### 字符串拼接
- 方法：用“+”号将数据进行拼接
- 目的：数据整合
- 只有字符串与字符串才能拼接

###### type() 函数
- 作用：查询数据类型
- 示例：print(type('查询的内容'))

###### 数据转换
- str() ：将其他数据类型转成字符串，也可用引号转换
- int() ：将其他数据类型转成整数
    1. 文字类和小数类字符串，无法转化成整数
    2. 浮点数转化成整数：抹零取整
- float() ：将其他数据类型转成浮点数
