# PPTist

- GitHub：https://github.com/pipipi-pikachu/PPTist
- 在线预览：https://pipipi-pikachu.github.io/PPTist

PPTist，一个基于 Vue3 的在线演示文稿PPT应用插件，支持文字、图形、图表、表格、公式、音视频等多种类型元素的添加，可以在浏览器中进行导入、编辑、演示、导出PPT等操作，提供多个右键菜单、快捷方式，方便使用
