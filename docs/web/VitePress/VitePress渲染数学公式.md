# vitepress 渲染数学公式

`markdown-it-mathjax3`：一个从 markdown-it-katex 改造而来、额外增加了 MathJax v3 和 SVG 渲染支持的插件。

## 安装

```sh
npm install markdown-it-mathjax3 -D
```

```
$\int\limits_{1}^{3}\frac{e^3/x}{x^2}\, dx$
```
