## 地图源数据

- 下载数据


[阿里云数据可视化平台](http://datav.aliyun.com/portal/school/atlas/area_selector)

- 合并数据

[geojson.io](https://geojson.io/#map=2/30.15/103.17)

左上角：

- Open ：上传已经下载的 json 文件，可以是多个 json 文件
- Save ：选择 `GeoJSON` 格式下载，然后改为 json 文件即可
