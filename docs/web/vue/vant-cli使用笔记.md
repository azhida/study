# vant-cli 使用笔记

我使用的环境：
- node:16.20.0
- npm:8.19.4


## 搭建

```sh
npm create vant-cli-app
# 按提示输入包名称：demo
# 选择 Vue 版本：Vue 3
# 选择 CSS 预处理器：Sass
# 看到 Successfully created demo 说名创建成功

cd demo
npm i
npm run dev
```
访问 http://localhost:5173/ 即可看到组件库主页了。
