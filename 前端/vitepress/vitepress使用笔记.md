# vitepress 使用笔记

Vitepress是使用Vue3+Vite来快速搭建一个个人网站的工具，网站搭建者不需要掌握Vue3，Vite等的具体内容，只需要简单的配置就可以生成Vue风格的个人网站

官方地址：https://vitejs.cn/vitepress/


## 安装

我的 node 版本：16.17.0

```
mkdir vitepress
cd vitepress
npm init
npm install -D vitepress
```

创建 vitepress
```
npx vitepress init
```

更多看官网：https://vitepress.dev/guide/getting-started
