###### 汉化步骤
```
File --> Settings --> Plugins
# 搜索 Chin，选择 中文简体 即可
```

###### 调整字体大小
```
# File --> Settings --> Keymap --> Editor Actions
# 找到如下选项，并双击
Decrease Font Size
Increase Font Size
# 选择 Add Mouse Shortcut，按住 Ctrl ，鼠标滚轮上下滚动即可
```
